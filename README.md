# Restaurant Reservation Management System

A production-ready full-stack application for managing restaurant table reservations. 

## Features
- **Customer Portal**: Book tables, edit, and cancel reservations.
- **Admin Dashboard**: Real-time operational analytics, restaurant floor layout map, and table provisioning.
- **Reservation Engine**: Robust engine handling capacities, table combinations, and overlapping times.

## Architecture
- **Backend**: Node.js, Express, MongoDB (Mongoose), JWT Auth.
- **Frontend**: React (Vite), React Router.

## Quick Start
1. Setup environment variables in `backend/.env`.
2. Run `npm run dev` to start both frontend and backend concurrently.

## Docker
Run `docker-compose up -d` to build and spin up the services via Docker.
